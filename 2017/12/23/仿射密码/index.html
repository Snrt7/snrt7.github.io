仿射密码

ekey(x) = (k1x + k2)mod26

dkey(y) = k1-1(y - k2)mod26

其中， k1-1表示 k1在Z26中的乘法逆，gcd(k1，26) = 1表示 k1与26互素。当 k1 = 1时，仿射密码变成了移位密码。
